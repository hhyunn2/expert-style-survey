안내문 및 동의 확인

이 설문은 이미지 스타일 평가 지표(style metric)를 만들기 전에, 사람이 어떤 기준으로 “같은 스타일” 또는 “다른 스타일”을 판단하는지 파악하기 위한 전문가 인터뷰입니다. 정답을 맞히는 테스트가 아니라 판단 이유와 근거를 수집합니다.

가능하면 이미지의 내용, 캐릭터, 포즈, 선명도, 완성도 자체보다는 “스타일”이라고 느껴지는 요소를 중심으로 답해주세요. 다만 내용이나 품질이 스타일 판단에 영향을 준다면 그것도 그대로 적어주세요.

기본 정보

A

스타일 판단 기준 자유 응답

초반에는 연구팀이 미리 정한 스타일 축(axis)을 공개하지 않습니다.

B

이미지 쌍(pair) 평가

각 이미지 쌍은 A/B로만 표시됩니다. 원본이 어느 쪽인지, 어떤 요소를 편집했는지는 먼저 공개하지 않습니다.

C

연구팀 스타일 축 및 편집 방식 검토

연구팀은 현재 스타일 변형 편집을 다섯 가지 축으로 나누어 보고 있습니다: 색감/팔레트, 선/외곽선 처리, 조명/명암, 질감/재질, 형태/비율.
